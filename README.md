# Hairven Salon & Spa Website

## Project Overview

Hairven Salon & Spa is a modern, responsive multi-page website designed for a fictional beauty salon and spa. The project showcases a professional online presence for a salon business by providing information about services, displaying a gallery of work, and allowing customers to contact the business or request appointments.

This project was developed as part of the ALX Front-End Capstone Project to demonstrate practical skills in HTML, CSS, and JavaScript while building a complete website from scratch.

The website focuses on clean design, responsiveness, and interactive user experience, simulating a real-world business website.

## Features

### Responsive Multi-Page Website

The website contains six main pages:

- Home
- Services
- Gallery
- About
- Contact
- Booking

Each page is designed with a consistent layout and responsive design that adapts to desktops, tablets, and mobile devices.

---

### Elegant and Minimal Design

The interface follows a minimal and premium visual style, using a structured layout, modern typography, and soft color tones to create a professional salon website experience.

---

### Testimonials Slider

The homepage includes a customer testimonial slider that automatically rotates through feedback while allowing users to manually navigate between testimonials.

Features include:

- Automatic slide transitions
- Previous and next controls
- Clean, centered layout

---

### Interactive Gallery with Lightbox

The gallery page includes an interactive lightbox viewer that allows users to view images in a larger format.

Lightbox functionality includes:

- Click to open enlarged image
- Next and previous image navigation
- Keyboard navigation (arrow keys)
- Escape key to close
- Click outside to close

---

### Contact Form with Validation

The contact page includes a form that allows users to send a message to the salon.

JavaScript validation ensures:

- All required fields are completed
- Email addresses are correctly formatted
- User-friendly success and error messages are displayed

---

### Booking Form

The booking page allows users to request salon appointments.

Users can provide:

- Name
- Phone number
- Email
- Service selection
- Appointment date
- Appointment time

Client-side validation ensures that all required fields are completed before submission.

---

### Google Maps Integration

The contact page includes an embedded Google Maps location, helping users easily locate the salon.

## Installation and Setup

To run the project locally:

### 1. Clone the repository

```
git clone https://github.com/eustaceyoc/hairven-salon-website.git
```

### 2. Navigate to the project directory

```
cd hairven-salon-website
```

### 3. Open the project

Open `index.html` in your browser.

You can also use a local development server such as VS Code Live Server for a better development experience.

---

## Usage

Once the website is opened in the browser, users can:

- Browse salon services
- View the gallery of work
- Read testimonials from customers
- Contact the salon through the contact form
- Request appointments through the booking form
